# TelegramBotAudioYT

## About
This telegram bot replies with full song audio file downloaded from YouTube by your search phrase.

In a nutshell, you can download any song by it`s name to your device for free using this nice bot. 
You can specify singer name after song name to refine search phrase and get more accurate result.


## API keys
Fill in bot name and bot token in application.yaml.

Fill in rapidapi key (you can get it here for free https://rapidapi.com/ytjar/api/youtube-mp3-download1/) and youtube api key for search requests in SongLoader class.
